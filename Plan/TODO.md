# 心理健康监测系统 TODO List

## 一、开发原则

1. 先完成 P0，保证系统主流程能跑通。
2. 再完成 P1，提升系统完整度。
3. 最后根据时间选择 P2。
4. 情绪识别先保证能调用成功，不追求高准确率。
5. 系统结果只作为辅助监测，不作为医学诊断。
